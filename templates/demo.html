We make media load super fast. We have a whole bunch of tricks up our sleeves to make your files smaller without sacrificing quality. When you upload a picture, we run it through lossless compression tools - they find the best way to make the file smaller, without reducing the quality of the file. And we have epsecially good tools for GIF files. If you upload a GIF to MediaCrush, we'll convert it into a video before we save it. You see, the GIF file format is quite old and quite bad. In the years since it came about, computers have become a lot better at compressing files. No, seriously - this example got 1600% faster when we converted it to a video:
